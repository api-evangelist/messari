# Messari (messari)

Messari provides crypto research, analytics, fundamentals, intelligence, and asset profiles through a REST API at `data.messari.io`. Endpoints cover asset metrics, market data, news, Token Unlocks, screener, governance, treasuries, and AI-powered Messari Copilot.
